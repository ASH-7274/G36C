# 超临界CO₂辅助聚氨酯喷涂技术

> 南林大团队项目 | AI助手梦火claw 协助维护

## 项目概述

超临界CO₂辅助聚氨酯喷涂技术是一种新型环保发泡技术，以超临界CO₂为物理发泡剂，替代传统CFC-11/HCFC-141b，减少VOCs排放，符合《蒙特利尔议定书》管控方向。

## 核心优势

- 成本降低约10%（vs 水发泡）
- 安全性更高
- 产品性能更优（隔热保温、防火、密度等指标经权威机构验证）
- 全程绿色环保

## 三大技术创新

1. **配方创新** — 优化多元醇/异氰酸酯配比，提升稳定性
2. **设备创新** — 团队自主研发，已获发明专利
3. **工艺创新** — 操作便利，工人上手快
