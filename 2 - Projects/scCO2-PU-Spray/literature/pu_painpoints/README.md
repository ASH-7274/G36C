## 四大痛点

1. **HCFC-141b淘汰压力**：2025年底死线，喷涂子行业剩余3000吨占总量34%
2. **替代技术困境**：碳氢需防爆改造、水发泡性能下降、HFCs本身也被管控
3. **VOC排放**：聚氨酯有机物挥发，环保政策收紧
4. **行业集中度低**：5000+企业分散，中小企业转型资金技术双不足

## 关键政策时间表

- 2025年12月：板材、喷涂、自结皮、鞋底全面禁用HCFC-141b
- 2026年1月：全行业禁止使用HCFC-141b发泡剂
- 2026年1月：禁止生产HFCs制冷剂家电产品
- 2024年冻结，2029年起HFCs开始削减
