## 内容亮点

### PP发泡综述（2025）
- 聚丙烯发泡的挑战：熔体强度低、弹性不足
- 四大改性方法：长链支化、交联、共混、填料
- 应用：汽车轻量化、隔热、阻燃、电磁屏蔽、油水分离

### PCL发泡技术（2024）
- ScCO₂发泡技术与生物医学结合
- 六种联合技术实现高性能PCL泡沫
- 骨组织工程、药物递送、医用敷料应用

### NIPU技术（2025）
- 非异氰酸酯聚氨酯：绿色环保替代传统PU
- 无异氰酸酯毒性，CO₂作为原料参与反应
- 2025年最新进展
