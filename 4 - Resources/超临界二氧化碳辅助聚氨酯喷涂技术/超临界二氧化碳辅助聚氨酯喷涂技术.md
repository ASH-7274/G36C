# 超临界二氧化碳辅助聚氨酯喷涂技术

## 条目说明

这是根据公开文献与网页资料整理的一版知识库条目，聚焦“超临界二氧化碳（scCO2）辅助聚氨酯喷涂/涂层/喷涂发泡”方向。

## 核心结论

- scCO2 的关键作用，是部分替代传统有机溶剂，降低体系黏度，改善喷嘴内流动与雾化行为。
- 该路线最直接的工程价值，是降低 VOC / HAP 排放，并改善喷涂均匀性。
- 对聚氨酯体系而言，目前公开资料里最直接相关的两条线是：
  - 聚氨酯树脂/涂层喷涂
  - 硬质聚氨酯喷涂发泡装备
- 当前可见公开研究中，喷嘴流场、入口压力、CO2 体积分数与喷雾均匀性的关系，是比较核心的分析点。

## 重点文献

### 1. Numerical Simulation of Flow Characteristics for Supercritical CO2-Sprayed Polyurethane Resin（2024）
- 来源：PubMed / Polymers
- 链接：<https://pubmed.ncbi.nlm.nih.gov/38611196/>
- 中文要点：通过 CFD 模拟 scCO2 喷涂聚氨酯树脂过程，分析入口压力和 CO2 体积分数对喷嘴内流场与喷雾液滴分布的影响。结果表明，更高 CO2 体积分数会降低密度并提高喷射速度；更高入口压力有助于获得更均匀的液滴分布。

### 2. UNICARB® Supercritical CO2 Coating Spray System
- 来源：P2 InfoHouse
- 链接：<https://p2infohouse.org/ref/20/19926/P2_Opportunity_Handbook/5_17.html>
- 中文要点：UNICARB 工艺利用超临界 CO2 替代部分有机溶剂，在保持可喷涂性的同时降低 VOC 排放，是理解该方向工业化价值的重要背景材料。

### 3. Supercritical Fluid Spray Application Technology
- 来源：P2 InfoHouse
- 链接：<https://p2infohouse.org/ref/25/24795.pdf>
- 中文要点：该资料系统介绍超临界流体喷涂技术的一般工艺原理，指出采用超临界 CO2 可在许多场景下降低约 70% 的溶剂相关 VOC。

### 4. 超临界二氧化碳辅助水发泡硬质聚氨酯喷涂装备及应用技术（2018）
- 来源：《聚氨酯工业》 / 维普
- 链接：<http://dianda.cqvip.com/Qikan/Article/Detail?id=674502140>
- 中文要点：该技术将水与二氧化碳按比例作为发泡剂，用于替代传统 HCFC 类物质，聚焦硬质聚氨酯喷涂装备与应用方向。
